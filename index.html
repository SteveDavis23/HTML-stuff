Presentasi HTML

Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language. HTML digunakan untuk membuat kerangka sebuah halaman web.

Apa yang Bisa Dilakukan HTML?

  • Menampilkan teks (judul, paragraf)
  • Menambahkan gambar
  • Membuat tautan (link)
  • Membuat daftar (list)
  • Membuat tabel
  • Membuat formulir (form)
  • Menambahkan media (video, musik, peta)

Apa yang Tidak Bisa Dilakukan HTML?

HTML tidak bisa membuat tampilan menjadi cantik. Itu tugas CSS.

HTML juga tidak bisa membuat halaman menjadi interaktif. Itu tugas JavaScript.
